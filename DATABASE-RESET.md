# Пересоздание базы данных

Для полной пересборки БД с новыми справочниками:

1. **PostgreSQL**: Удалите базу или пересоздайте её:
   ```sql
   DROP DATABASE IF EXISTS biobank;
   CREATE DATABASE biobank;
   ```

2. **application.yaml**: Временно смените `ddl-auto`:
   ```yaml
   jpa:
     hibernate:
       ddl-auto: create  # или create-drop
   ```

3. Запустите backend — Hibernate создаст схему, затем выполнится `data.sql` с начальными данными.

4. После успешного запуска верните `ddl-auto: update` для дальнейшей работы.

## Значения по умолчанию после пересборки

- **Гендер пациента** (ENUM): UNKNOWN, MALE, FEMALE (DEFAULT: UNKNOWN)
- **Статус срока годности** (ENUM): GREEN, YELLOW, RED (DEFAULT: GREEN)
- **Национальности**: Армянин, Азербайджанец, Белорус, Бурят, Немец, Русский, Украинец, Не указано
- **Статусы образцов**: В хранилище, Изъят, Утилизирован, Исчерпан
- **Типы образцов**: Кровь, Плазма, Сыворотка, Моча, Кал, Биопсия, Лейкоциты, гемолизаты, ДНК, РНК, Десневая жидкость, Слюна, Парафиновые блоки, Гистологические стёкла, Волосы (с иконками)
- **Шаблоны контейнеров**: 6 предустановленных (криобоксы 10×10, 9×9; штативы 10×5, 8×12, 5×5; транспортная стойка 8×5)
